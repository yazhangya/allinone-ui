## 自动生成代码的展示界面

1. 该项目的节本架构来自[angular-start](https://github.com/AngularClass/angular-starter.git) 
2. 该项目其基本想法是实现自动话生成项目所用的基本代码和页面

## 交互逻辑

1. 选择schema
2. 选择相应的表结构
3. 对相应的字段进行选择是否列表展示或者表单填写
4. 前端交互使用angular+material来实现
